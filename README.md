
## Description
1. Seed file for wallets is located in `/seed-db/Wallet.yml`

2. Database connection is located in `/ormconfig.json`

3. Postman collection is located in `/custodial-poc.postman_collection.json`

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
